# Desk-CLI

## Installation
```
brew install pipx
pipx ensurepath
pipx install idasen
```

## Usage
```
~ desk check
Height 80cm is within the acceptable range
~ desk sit  
Moving desk to sit position...
Desk moved to sit position successfully.
~ desk stand
Moving desk to stand position...
Desk moved to stand position successfully.
```
